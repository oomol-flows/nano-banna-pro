## 工作原理

Nano Banana Pro 为你提供了两个可以协同工作的构建模块:

### 1. 图像生成器

这是魔法开始的地方。你提供想要创建内容的描述,生成器就开始制作你的图片。它会给你一个追踪编号,这样你就能查看进度。

### 2. 结果查询器

这个模块会监视你的图片创建过程,并在准备好时通知你。它会自动等待图片完成,所以你不需要反复检查。

**协同工作:**
这两个模块被设计为相互配合。你可以考虑将图像生成器连接到结果查询器,打造流畅的工作流程 - 生成器创建你的追踪编号,查询器使用它来监控你的图片直到完成。当你想要免手动操作的体验时,这种组合通常效果很好。

当然,你也可以单独使用每个模块,或以你自己的创意方式将它们与其他工具组合。选择权在你手中。

## 开始使用

### 你的第一张图片

1. 从**图像生成器**模块开始
2. 写下你想要创建内容的描述
3. 将它连接到**结果查询器**模块
4. 让它运行,等待你的图片出现

就是这样!你的图片将准备好供查看和使用。

### 获得更好结果的技巧

**写清晰的描述:**
不要只写"一只狗",试试"一只金毛寻回犬幼犬在阳光明媚的公园里玩耍"。更多细节有助于创建你想象中的确切画面。

**从简单开始:**
在尝试想法时,使用更快的质量设置。一旦你对结果满意,就可以用更高质量重新创建。

**选择合适的形状:**
想想你将在哪里使用图片。手机屏幕最适合竖版图片,而电脑屏幕更喜欢横版图片。

**对高质量保持耐心:**
更高质量的图片需要更长时间来创建,但结果是值得等待的。如果你需要快速获得图片,请相应地做好计划。

## 需要帮助?

**图片没有出现?**
- 确保你的描述清晰且详细
- 检查你是否连接到互联网
- 先尝试一个更简单的图片来测试系统

**花费时间太长?**
- 高质量图片自然需要更多时间
- 如果需要,你可以在设置中增加等待时间
- 从较低质量开始可以帮助你更快预览

**有什么不工作?**
- 仔细检查所有模块是否正确连接
- 确保你已提供所有必需的信息
- 尝试重新启动工作流程
